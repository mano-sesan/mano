# Application Android - Expo x React Native

## Running the app

### The first time - or when a new native dependencie is installed

- Install the dependencies:

  ```sh
  npx expo install
  ```

- Prebuild first

  ```sh
  yarn prebuild
  ```

- Build and run Android development builds:

  ```sh
  yarn build-dev-android
  ```

### Development server

- Build and run Android development builds:

  ```sh
  yarn start
  ```
